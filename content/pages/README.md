# Website Page Map

This folder contains content drafts for future pages planned around the PloverCRM website after the homepage.

## Priority order

1. Use case pages
2. Integration pages
3. Comparison pages, starting with the WordPress CRM vs SaaS CRM pillar
4. Pricing and demo support pages
5. Resource pages
6. Company, support, and legal pages

## Folder groups

- [use-cases/](use-cases/) — audience and business-type pages.
- [integrations/](integrations/) — source, follow-up, and outcome integration pages.
- [comparisons/](comparisons/) — focused educational comparison pages, led by WordPress CRM vs SaaS CRM and forms/spreadsheets vs CRM.
- [resources/](resources/) — research, guides, and educational content.
- [support/](support/) — docs, demo, onboarding, and FAQ support pages.
- [company/](company/) — about and trust content.
- [legal/](legal/) — privacy, terms, and policy pages.
